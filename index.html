2023 - SEFT

PTC primer año - SEFT

SEFT es un programa desarrollado en C# con el propósito principal de facilitar la administración de una tienda de videojuegos. Este sistema está diseñado para gestionar de manera eficiente el inventario, las ventas, las compras y otros aspectos claves de la operación de la tienda. A través de una interfaz intuitiva, SEFT permite a los usuarios acceder a funciones como la gestión de productos (videojuegos, consolas y accesorios), control de stock, generación de facturas y reportes de ventas, y la actualización de precios o descuentos. Para mejorar la accesibilidad y funcionalidad del programa, también se implementaron páginas web que permiten a los clientes consultar el catálogo de productos disponibles, conocer las promociones vigentes y realizar compras en línea. Estas páginas web fueron desarrolladas utilizando tecnologías como HTML, CSS, JavaScript y Bootstrap, lo que garantiza una experiencia de usuario optimizada tanto en computadoras de escritorio como en dispositivos móviles. En resumen, SEFT no solo se limita a ser un software para la gestión interna de la tienda, sino que también ofrece una plataforma de comercio electrónico moderna que integra la administración de la tienda física con la posibilidad de realizar ventas en línea, mejorando la eficiencia operativa y ampliando el alcance de la tienda.

2024 - TABLETAP

PTC segundo año - TABLETAP

TableTap es una aplicación móvil desarrollada utilizando Android Studio y Java, diseñada específicamente para la administración de un restaurante. Este sistema busca optimizar los procesos de atención al cliente, gestión de mesas, pedidos y facturación en establecimientos de comida. Con TableTap, los empleados del restaurante pueden asignar mesas, tomar pedidos desde dispositivos móviles, y enviarlos directamente a la cocina o al sistema de facturación, todo en tiempo real. El sistema también permite gestionar el inventario de los ingredientes utilizados en el menú, asegurando que los productos estén siempre disponibles y evitando el desperdicio de alimentos. Además, TableTap proporciona reportes detallados sobre las ventas, el rendimiento de los empleados y las preferencias de los clientes, lo que permite a los gerentes del restaurante tomar decisiones informadas para mejorar el servicio y la rentabilidad del negocio. Para complementar la experiencia del usuario, TableTap también cuenta con páginas web que se desarrollaron utilizando HTML, CSS, JavaScript y Bootstrap.
